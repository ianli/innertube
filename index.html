About

Blurb

The Innertube Widgets API is used to create widgets of personal informatics visualizations for users to add to their Innertube Dashboard.

Dependencies

Innertube Widgets API is dependent on these great open-source projects:

  • easyXDM for cross-domain messaging
  • Underscore.js for utility functions
